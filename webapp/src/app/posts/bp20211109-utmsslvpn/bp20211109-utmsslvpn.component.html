Sophos UTM (SG-Series) -- SSL VPN

Besides IPSec Sophos UTM (SG-Series) provides SSL-VPN as well. This post describes how one can change the target host. For some reason we rather used a SSL-VPN than IPSec.

Problem - IP-Adress changed

Due to some reorganisation I had to reroute a SSL-VPN connection to a different host.

Solution

In the config of the SSL click this button and select your new (DNS)Host.
